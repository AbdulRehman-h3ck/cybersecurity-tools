# 📖 How to Use: APK Forensic Intel

This guide will help you get started with the APK Forensic Intelligence Tool, whether you are a developer or a casual user.

---

## 🛠️ Option 1: Using the Standalone EXE (Recommended for Windows)

If you don't want to install Python, follow these steps:

1.  Go to the **Releases** section on the main page of this repository.
2.  Download the latest `APK_Forensic_Lab.exe`.
3.  Double-click the file to launch the GUI.
4.  Click **Browse APK** to select the file you want to analyze.
5.  Press **START DEEP ANALYSIS** and wait for the results to appear in the green console.
