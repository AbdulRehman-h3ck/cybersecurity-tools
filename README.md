# Cybersecurity Tools

This repository contains a collection of cybersecurity tools that I am building as part of my learning journey in ethical hacking.

Instead of only using existing tools, I focus on creating my own tools to understand how things work internally.

## Purpose

The main goal of this repository is to:
- Learn core cybersecurity concepts by building tools
- Automate common tasks used during penetration testing
- Improve problem-solving and scripting skills

## Tools Included

- Mini Recon Tool (information gathering)
- Port Scanner (network scanning)
- Password Checker (basic security validation)

Each tool is designed to be simple, practical, and focused on real-world use cases.

## How to Use

1. Navigate to the specific tool folder
2. Follow the instructions inside each tool's README file
3. Run the script using Python

## Why this matters

Building tools helps in understanding:
- How attackers think and operate
- How vulnerabilities are discovered
- How systems can be analyzed and tested

## Future Improvements

- Add more advanced tools
- Improve performance and accuracy
- Add CLI arguments and better output formatting

These tools are created for educational purposes only. Do not use them on systems without proper authorization.
